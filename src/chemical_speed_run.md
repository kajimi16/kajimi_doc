## 第一章：重点
### Chapter 1. 化学反应热

- $
U = Q + W \qquad Q_v = \Delta U \quad \text{(等容反应热)}
$
$
Q_p = H_2 - H_1 = \Delta H \quad \text{(等压反应热)} \qquad \text{定义 } H \equiv U + PV
$
- 赫斯定律 $
\Delta_r H_m = \sum_i \Delta_r H_{m,i}
$

- 标准态、标准摩尔生成焓的定义；化学反应进度 
-  $ \Delta_r H_m^\ominus = \sum_B v_B \Delta_f H_{m,B}^\ominus \qquad $
- 最稳定的单质 $ \quad \Delta_f H_{m,B}^\ominus = 0 \, \text{KJ} \cdot \text{mol}^{-1}   $

**赫斯定律（盖斯定律）表述：**
- 总反应的焓变等于各分步反应的焓变之和时，其成立的条件为任意过程；
- 总反应的反应热等于各分步反应的反应热之和时，其成立的条件是等温、等容、不做有用功或者等温、等压不做有用功。


## 第二章：重点
### Chapter 2. 化学反应进行的方向与限度
- 化学反应方向的判断 
- 
$ \Delta G = \Delta H - T \Delta S \leq W_{有} $ &nbsp;&nbsp; 

$ \Delta G \leq 0   $  

$ \Delta_f G^\theta_{m,B} KJ\cdot mol^{-1}  $
- 化学反应标准吉布斯自由能变化的计算  

$
\Delta_r G_m^\ominus = \Delta_r H_m^\ominus - T \Delta_r S_m^\ominus
$  

$
\Delta_r G_m^\ominus = \sum_B v_B \Delta_f G_{m,B}^\ominus
$

$
T = \frac{\Delta_r H_m^\ominus}{\Delta_r S_m^\ominus}
$ 
- 范托夫等温方程式

$
\Delta_r G_m = \Delta_r G_m^\ominus + RT \ln J
$

$
J = \frac{(p_G/p^\ominus)^g (p_D/p^\ominus)^d}{(p_A/p^\ominus)^a (p_F/p^\ominus)^f}
$
- 化学反应平衡常数的定义

$
K^\ominus = \frac{\left(\frac{p_G}{p^\ominus}\right)^g \left(\frac{p_D}{p^\ominus}\right)^d}{\left(\frac{p_A}{p^\ominus}\right)^a \left(\frac{p_F}{p^\ominus}\right)^f}
$  
$
K^\ominus = \frac{\left(\frac{c_G}{c^\ominus}\right)^g \left(\frac{c_D}{c^\ominus}\right)^d}{\left(\frac{c_A}{c^\ominus}\right)^a \left(\frac{c_F}{c^\ominus}\right)^f}
$  
- 影响化学平衡的因素：温度、浓度和压力。

- 化学反应平衡常数的计算

$
\Delta_{r} G_{m}^\ominus = -RT \ln K^\ominus
$
- 化学反应平衡常数的计算与反应方向的判断
  
$
\Delta_{r} G_{m} = -RT \ln K^\ominus + RT \ln J
$
- 范特霍夫公式
  
$
\ln \left( \frac{K^\ominus(T_2)}{K^\ominus(T_1)} \right) = \frac{\Delta_r H_m^\ominus}{R} \left( \frac{1}{T_1} - \frac{1}{T_2} \right)
$


## 第三章：重点
### Chapter 3. 化学反应速率
- 质量作用定律
  
$
r = \frac{1}{\nu_B} \cdot \frac{dc(B)}{dt} = k [c(A)]^\alpha [c(F)]^\beta
$
- 一级反应
  
$
\ln \frac{c(B)_t}{c(B)_0} = -kt
$

$
t_{1/2} = \frac{0.693}{k}
$
- 影响化学反应速率的因素、化学反应速率常数的因素
  
- 阿伦乌尼斯公式

$
k = Ae^{-\frac{E_a}{RT}}
$

$
\ln \frac{k_2}{k_1} = \frac{E_a}{R} \left( \frac{1}{T_1} - \frac{1}{T_2} \right) = \frac{E_a}{R} \left( \frac{T_2 - T_1}{T_1 T_2} \right)
$
- 催化剂的作用

$
\Delta_r G_m = -zFE
$

$
\Delta_r G_m^\ominus = -zFE^\ominus
$
